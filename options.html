Informations sur l'extension ChatGPT Sync Manager

Description

ChatGPT Sync Manager est une extension Chrome qui vous permet de synchroniser et gérer facilement vos GPTs personnalisés et vos conversations ChatGPT. Avec une interface moderne et intuitive, gardez tous vos outils IA organisés et accessibles.

Fonctionnalités principales

  • Synchronisation automatique de vos GPTs
  • Gestion des conversations ChatGPT
  • Catégorisation intelligente
  • Interface moderne avec thème sombre/clair
  • Export/Import des données
  • Statistiques d'utilisation détaillées
